Authlib Loginpass
=================

Social connections powered by [Authlib][]. This library is a part of Authlib project.
It works well with Authlib v0.7+.

[Authlib]: https://authlib.org/


Features
--------

Connections that Authlib Loginpass contains:

- [x] Google
- [x] GitHub
- [x] Gitlab
- [x] Twitter
- [x] Facebook
- [x] Dropbox
- [x] Reddit
- [ ] Linkedin
- [ ] Azure
- [ ] Discord
- [ ] Slack
- [ ] Jira
- [ ] Douban
